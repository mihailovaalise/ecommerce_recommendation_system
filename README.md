# ecommerce_recommendation_system
Курсовая работа: "Построение рекомендательной системы для интернет-магазина одежды". Система использует многомодальные признаки товаров (текстовые, категориальные, визуальные)  для поиска похожих товаров и формирования рекомендаций. Включает интерфейс на Streamlit и  метрики качества рекомендаций.
